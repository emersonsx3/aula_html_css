ID e CLASS

ID

o id funciona como ter uim CPF. assim como so posso ter um numero de cpf cada elemento so pode ter um valor de id. oCPF é único, ou seja, ninguém pode ter CPF IGUAL AO MEU. Isso significa que o ID também é único, então não é possivel repetir o mesmo valor duas vezes

CLASS

O CLASS funciona de maneira oposta ao ID. Diferente do ID em que você pode ter vários valores. também diferente do ID, em que não se pode repetir um valor, no CLASS você pode utulizar o mesmo valor para vários elementos.
